# UK-Hypertension
I created a synthetised UK hypertension dataset in Excel and turned it into an interactive dashboard. I used pivot tables, slicers, and charts to explore how blood pressure varies by age group, sex, residence, lifestyle factors, and treatment type. 
The goal of this project is to explore how hypertension patterns vary across demographics, lifestyle factors, and treatments using a synthetised UK dataset I built in Excel, and to practise turning clinical questions into data‑driven insights.

📊 Key Insights

Average blood pressure differs across age groups, sex, and residence (urban, rural, suburban), highlighting how context influences hypertension risk.

Treatment type (single vs combination therapy) shows different average blood pressure and control status, suggesting opportunities to review medication strategies.

Lifestyle factors such as smoking, physical activity, and dietary habits are associated with changes in average systolic and diastolic blood pressure, reinforcing the importance of prevention and patient education.

City‑level averages help quickly spot locations with higher blood pressure values, which could guide targeted interventions in a real‑world setting.

📈 Dashboard Preview

The Excel dashboard includes:

KPI cards for average systolic and diastolic BP

Pivot tables and charts comparing age, sex, residence, and lifestyle factors

Visuals showing treatment type vs control status

A map‑style or city comparison view for average BP by UK location

🛠️ Tools Used

Excel (data cleaning, formulas, visualisation)

Pivot Tables & Pivot Charts (aggregation and comparisons)

Slicers (interactive filtering by age group, sex, residence, lifestyle, treatment)

📂 Data Source

Synthetised hypertension dataset representing UK adults

Variables include: age, sex, residence, income, comorbidities, lifestyle factors, blood pressure readings, and treatment type

🔍 How to Explore the Dashboard

Use slicers to filter by age group, sex, residence, and lifestyle factors

Hover over charts to see exact blood pressure values and group differences

Switch between treatment views (single vs combination therapy) to compare control status
